Accessibility Statement

Last Updated: July 2026

We are committed to ensuring digital accessibility for people with disabilities, in accordance with the Israeli Equal Rights for Persons with Disabilities Law (1998), its accompanying Accessibility Regulations (2013), and the European Standard EN 301 549.

Compliance Status and Standard

This application conforms to the Web Content Accessibility Guidelines (WCAG) 2.1 published by the W3C at Level AA, as well as the Israeli Standard IS 5568 (Level AA).

Accessibility Features Implemented

  • Alternative Table View: Because interactive mapping and marker clustering present technical barriers for assistive technologies, the application includes an accessible table view. This table displays all underlying data (countries, partners, and algae types: MacroAlgae, MicroAlgae, and Cyanobacteria) and is fully optimized for screen reader navigation.
  • Keyboard Navigation: Full keyboard operation is supported using the Tab key for sequential focus navigation and Enter/Space to activate controls.
  • Visual Focus Indicators: Clear focus rings are implemented across all interactive elements to assist keyboard users.
  • Semantics and Scaling: Semantic HTML5 markup is used throughout, and the layout supports text scaling up to 200% without loss of functionality.

Exemptions (Interactive Maps)

According to accessibility regulations, complex map canvases and marker clusters are exempt from full accessibility requirements, provided an accessible text alternative is available. The integrated Table View serves as the compliant alternative to the interactive map display.

הצהרת נגישות

תאריך עדכון אחרון: יולי 2026

אנו פועלים להבטחת נגישות דיגיטלית לאנשים עם מוגבלות, בהתאם לדרישות חוק שוויון זכויות לאנשים עם מוגבלות (תשנ"ח-1998), תקנות שוויון זכויות לאנשים עם מוגבלות (התאמות נגישות לשירות), תשע"ג-2013, והתקן האירופי EN 301 549.

תקינות ורמת נגישות

האתר מותאם לעמידה בהנחיות הנגישות למידע באינטרנט (WCAG 2.1) של ארגון W3C ברמת התאמה AA, וכן בתקן הישראלי ת"י 5568 רמה AA.

אמצעי הנגישות שהוטמעו במערכת

  • תצוגת טבלה חלופית (Alternative Table View): מכיוון שמפה אינטראקטיבית וקיבוץ סמנים (Marker Clustering) מהווים אתגר טכני לטכנולוגיות מסייעות, המערכת כוללת תצוגת טבלה נגישה המציגה את מלוא הנתונים (מדינות, שותפים וסוגי אצות: MacroAlgae, MicroAlgae, Cyanobacteria). הטבלה מותאמת באופן מלא לקריאה באמצעות קוראי מסך.
  • ניווט מקלדת: תמיכה בניווט מלא באמצעות מקש Tab ומעבר לוגי בין רכיבים, ושימוש ב-Enter או Space להפעלת אלמנטים אינטראקטיביים.
  • חיווי ויזואלי (Focus Indicators): לכל רכיב אינטראקטיבי במערכת קיים חיווי פוקוס ברור ונראה לעין בעת ניווט באמצעות מקלדת.
  • סמנטיקה והגדלה: שימוש בתגיות HTML5 סמנטיות ותמיכה בשינוי גודל תצוגה בדפדפן עד 200% ללא פגיעה בפונקציונליות או אובדן מידע.
