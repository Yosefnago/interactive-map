Accessibility Statement

Efforts have been made to provide an accessible user experience on this website in accordance with commonly accepted accessibility principles.

The website includes an accessible table view that provides full access to the information presented in the interactive map for keyboard and screen-reader users.

The site supports keyboard navigation, visible focus indicators, and a structured content layout.

The website was designed primarily for desktop use.
